# Souvik Biswas — Portfolio

Personal portfolio website built with vanilla HTML, CSS, and JavaScript.

🌐 **Live:** [souvik-biswas.netlify.app](https://souvik-biswas.netlify.app/)

---

## Sections

- **Home** — Intro, bio, and links to resume, GitHub, LinkedIn, LeetCode
- **Timeline** — Work experience and education
- **Portfolio** — Featured projects
- **Achievements** — LeetCode contest rank and academic highlights
- **Contact** — Location, email, phone, and social links

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Font Awesome 6 for icons
- LordIcon for animated icons
- Intersection Observer API for scroll animations
- Formspree for contact form (currently disabled)
